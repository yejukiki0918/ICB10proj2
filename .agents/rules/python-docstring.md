# 파이썬 코드 작성 규칙: Docstring 필구

파이썬 파일(`.py`)을 새로 작성하거나 수정할 때는 반드시 파일 최상단에 해당 파일의 목적과 내용을 설명하는 docstring을 한국어로 작성해야 합니다.

## 작성 형식

- 파일의 맨 첫 줄부터 시작합니다.
- 삼중 따옴표(`"""`)를 사용합니다.
- 한국어로 작성합니다.
- 파일의 전반적인 역할, 주요 기능, 작성자(필요시), 생성일(필요시) 등을 포함합니다.


이 규칙은 모든 파이썬 파일에 적용되며, 코드의 가독성과 유지보수성을 높이기 위해 반드시 준수해야 합니다.
