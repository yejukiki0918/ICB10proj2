# Streamlit Dashboard Validation Checklists

## 1. 단일 항목 분포 및 기술 통계 검증 (10항목)
각 개별 데이터 항목(변수)이 가진 특성을 얼마나 정확하게 파악하고 있는지를 검증합니다.
1. **대표값의 적절성**: 평균(Mean)뿐만 아니라 중앙값(Median), 최빈값(Mode)을 병기하여 이상치에 의한 왜곡을 확인하는가?
2. **결측치 가시성**: 항목별 결측치(Null) 비율을 명시하고, 분석 시 제외/대체(Imputation) 방식이 투명한가?
3. **분포의 비대칭성**: 왜도(Skewness)와 첨도(Kurtosis)를 통해 데이터가 한쪽으로 쏠려 있는지 정량적으로 제시하는가?
4. **이상치(Outlier) 식별**: Box Plot이나 IQR(Interquartile Range) 분석을 통해 극단값을 통계적으로 정의하고 보여주는가?
5. **범주형 균형 분석**: 범주형 항목의 경우, 특정 클래스에 데이터가 과도하게 쏠린 불균형(Imbalance) 상태를 경고하는가?
6. **데이터 범위 유효성**: 최소값(Min)과 최대값(Max)이 비즈니스 도메인 상의 논리적 범위(예: 나이가 음수 등)를 벗어나지 않는가?
7. **변동성 측정**: 표준편차와 변동 계수(CV)를 통해 항목 간의 상대적인 안정성을 비교할 수 있는가?
8. **데이터 타입 정합성**: 수치형, 범주형, 날짜형 데이터가 분석 목적에 맞게 올바르게 캐스팅되어 처리되는가?
9. **시간적 추세(Trend)**: 시계열 데이터의 경우, 개별 항목의 시간에 따른 변화량과 성장률을 계산하여 보여주는가?
10. **집중도 분석**: 4분위수(Percentile)를 통해 상위 10%, 25% 등의 데이터가 어디에 집중되어 있는지 파악 가능한가?

## 2. 항목 간 관계 및 상관성 분석 검증 (10항목)
여러 항목이 서로 어떻게 얽혀 있는지, 그 관계의 강도와 패턴을 검증합니다.
11. **상관행렬(Correlation Matrix)**: 피어슨(Pearson) 또는 스피어먼(Spearman) 상관계수를 통해 변수 간 선형/비선형 관계를 제시하는가?
12. **다중공선성 경고**: 상관계수가 지나치게 높은(예: 0.9 이상) 변수들 간의 중복 분석 위험을 사용자에게 알리는가?
13. **산점도(Scatter Plot) 패턴**: 두 수치형 항목 간의 군집화 경향이나 비선형적 관계를 시각적으로 확인할 수 있는가?
14. **그룹별 평균 비교**: 특정 범주형 항목에 따른 수치형 항목의 차이(예: 성별에 따른 구매액)를 통계적으로 유의미하게 비교하는가?
15. **다차원 히트맵**: 3개 이상의 항목 관계를 히트맵이나 버블 차트로 효율적으로 시각화하고 있는가?
16. **인과관계 추론 보조**: 시차 분석(Lag Analysis) 등을 통해 특정 항목의 변화가 사후에 다른 항목에 미치는 영향을 추적하는가?
17. **영향력 분석(Feature Importance)**: 목표 변수(Target)에 가장 큰 영향을 주는 핵심 항목들을 우선순위대로 보여주는가?
18. **파레토 분석(80/20)**: 소수의 핵심 항목이 전체 결과의 80%를 차지하는지 등의 집중도 분석이 포함되어 있는가?
19. **복합 지표(Index) 생성**: 여러 항목을 결합하여 분석 목적에 맞는 가중치 합산 지표를 산출하고 검증하는가?
20. **상호작용 필터링**: 사용자가 특정 항목을 필터링했을 때, 연관된 다른 항목들의 통계값이 실시간으로 동기화되는가?

## 3. 고급 기법 및 시스템 무결성 검증 (10항목)
분석 결과의 신뢰성과 사용자에게 전달되는 인사이트의 질을 검증합니다.
21. **텍스트 항목 분석**: 비정형 데이터(텍스트)가 있다면 워드클라우드나 TF-IDF를 통해 핵심 키워드를 추출하는가?
22. **차원 축소 시각화**: 항목이 너무 많을 때 PCA(주성분 분석) 등을 통해 저차원에서 데이터 구조를 보여주는가?
23. **군집 분석(Clustering)**: 항목들의 유사도를 기반으로 유사한 관측치끼리 묶어서 패턴을 발견하도록 돕는가?
24. **가설 검정 지원**: 분석 결과가 단순 우연인지, 통계적 유의성(p-value 등 개념)을 참고할 수 있는 장치가 있는가?
25. **데이터 최신성**: 대시보드에 로드된 데이터의 마지막 업데이트 시점과 데이터 소스의 신뢰성이 명시되어 있는가?
26. **드릴 다운(Drill-down)**: 요약된 통계 항목을 클릭했을 때 실제 상세 데이터 항목으로 접근하여 검증할 수 있는가?
27. **What-if 시뮬레이션**: 특정 항목의 값을 임의로 조정했을 때 전체 지표가 어떻게 변하는지 시나리오 분석이 가능한가?
28. **에러 핸들링**: 잘못된 데이터 형식이나 극단적인 필터 조건 입력 시 시스템이 멈추지 않고 적절한 안내 메시지를 출력하는가?
29. **주석 및 가이드**: 각 분석 항목에 대한 툴팁(Tooltip)이나 설명이 제공되어 사용자의 오독을 방지하는가?
30. **보고서 자동 요약**: 분석된 항목들의 주요 발견 사항(Key Findings)을 텍스트로 요약하여 인사이트를 바로 제공하는가?
