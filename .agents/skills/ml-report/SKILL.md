# ML Report Creator Skill

이 스킬은 분류(Classification), 회귀(Regression), 군집화(Clustering) 등의 다양한 머신러닝 분석 요구사항에 대해, 데이터의 통계적 속성을 탐색하고 여러 모델을 학습시킨 뒤 다각도로 검증하는 고품질 마크다운 보고서를 자동으로 컴파일합니다.

## 핵심 요구사항 및 가이드라인

1. **머신러닝 모델 수 및 평가지표**:
   - 최소 **3개 이상의 모델**을 학습하고 성능을 비교해야 합니다.
     - 예: 의사결정나무(Decision Tree), 랜덤 포레스트(Random Forest), 그래디언트 부스팅(Gradient Boosting) 등
   - 모델 성능 검증을 위해 최소 **5개 이상의 평가지표**를 산출하고 표로 정리해야 합니다.
     - 분류 예시: 정확도(Accuracy), 정밀도(Precision), 재현율(Recall), F1-Score, ROC AUC

2. **시각화와 표의 동반 구성**:
   - 모든 시각화 결과는 그래프 이미지 단독으로 두지 않고, 반드시 대응하는 **데이터 분석 표**를 쌍으로 기재합니다.
     - 막대그래프: 피봇테이블(Pivot Table) 또는 교차표(Cross-tabulation)
     - 히스토그램/박스플롯: 변수별 상세 기술통계량 표(대표값, 이상치 경계값 등)
   - 시각화 시 한글 깨짐을 방지하기 위해 `koreanize-matplotlib` 라이브러리를 임포트하여 사용합니다.

3. **고품질 인사이트 및 액션 플랜**:
   - 각 분석 단계 및 시각화 결과에 대해 최소 **500자 이상의 정밀 분석 인사이트**를 한국어로 기술합니다.
   - 단순한 통계값 나열이 아닌, 데이터 이면에 숨겨진 의미와 이를 비즈니스/마케팅 현업에 직접적으로 투영할 수 있는 액션 플랜(Action Plan)을 논리적으로 도출합니다.

4. **오류 없는 Mermaid 다이어그램**:
   - 파이프라인 흐름이나 데이터 흐름, 프로세스 진행 순서에 대해 Mermaid.js 구문을 활용해 시각화 요소를 추가합니다.
   - 마크다운 내에서 깨지지 않고 완벽하게 렌더링되도록 괄호와 문자를 큰따옴표로 이스케이프 처리합니다.

## 스킬 파일 구성 및 구조

```
ml-report/
├── SKILL.md (본 지침서)
└── scripts/
    └── generate_ml_report.py (분석 및 리포트 생성을 처리하는 공용 파이썬 자동화 스크립트)
```

## 스크립트 실행 지침

- 분석 대상 데이터셋 경로, 타겟 컬럼명, 문제 유형(분류/회귀/군집화 등), 리포트 출력 경로를 인자로 받아 `generate_ml_report.py`를 실행할 수 있습니다.
- 스크립트 내부에서 모델링 및 평가지표 연산을 거쳐 리포트 마크다운 문서 및 matplotlib 차트 이미지들을 지정된 디렉토리에 물리적으로 저장하도록 유도합니다.
