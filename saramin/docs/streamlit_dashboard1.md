### 1. 프로젝트 개요 및 컨셉
- 제목: 취업 시장의 자격증 미스매치(Gap) 분석 및 지원 적합도 자가진단 대시보드
- 핵심 목적: 구직자의 자격증 검색량(공급)과 기업 채용 공고의 자격증 언급 빈도(수요)의 격차를 시각화하고, 구직자의 보유 자격증과 기업 요구사항을 매칭해 주는 B2B/B2C 겸용 가성비 솔루션 구현

### 2. 가상 데이터 셋 구조 (코드 내에 Mock Data로 포함해 줄 것)
실제 DB 연결 전 데모 테스트를 위해 코드 상단에 Pandas DataFrame 형태로 가상 데이터를 정의해 주세요.
1) 공고 데이터 (df_jobs): 
   - 컬럼: 공고ID, 기업명, 직무(인사, 회계, 감사, 데이터분석), 우대자격증(리스트 또는 쉼표로 구분된 문자열. 예: "SQLD, 정보처리기사")
2) 자격증 통계 데이터 (df_licenses):
   - 컬럼: 자격증명, 구직자_월간검색량, 실제_공고_등장수

#### 💻 탭 1: 구직자 탭 (스펙 자가진단)
1. 유저가 자신의 보유 자격증을 다중 선택할 수 있는 st.multiselect 구현 (예: 정보처리기사, SQLD, 컴퓨터활용능력, 토익 등)
2. 진단 버튼(st.button)을 누르면 내부 매칭 로직 가동:
   - 파이썬의 `in` 연산이나 단순 집합(set) 비교를 활용하여, 사이드바에서 선택한 직무의 공고들이 요구하는 자격증과 유저가 보유한 자격증의 '일치율(%)'을 산출합니다.
   - 계산 공식: (유저 자격증 중 공고 우대사항에 포함된 개수 / 해당 직무 공고들의 평균 필요 자격증 수) * 100
3. 결과 화면:
   - st.metric을 활용해 "내 자격요건 일치율: XX%"를 크고 직관적으로 표시
   - 해당 직무 공고들에서 가장 많이 등장하는 자격증 순위 TOP 3를 리스트나 작은 테이블로 보여주며 "나에게 지금 부족한 추천 스펙" 가이드 제시

#### 🏢 탭 2: 인사팀 탭 (시장 미스매치 분석)
1. 사이드바에서 선택한 직무의 '구직자 자격증 관심도(검색량)'와 '실제 기업 우대 빈도(공고 등장수)'를 비교하는 바 차트(st.bar_chart 또는 plotly 등) 시각화
2. 인사팀을 위한 비즈니스 제언(st.info 또는 st.success) 텍스트 출력:
   - 예: "현재 감사 직무 구직자들은 '컴퓨터활용능력'을 많이 검색하지만, 실제 공고에서는 'CISA/CIA'의 수요가 압도적입니다. 양질의 지원자를 유입시키려면 공고 우대사항 태그에 구직자 검색 키워드를 의도적으로 배치하십시오."
